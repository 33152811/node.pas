# node.pas
Event-driven progmming for EMB Delphi, powered by libuv.

NodeJs-like ecosystem
---------------------   
* buffer (np.buffer.pas)
* from duvLib: 
  *  loop
  *  setTimeout
  *  setInterval
  *  setImmediate
  *  nextTick
  *  tcp
  *  pipe
  *  tty
  *  child process
* http server
* https server powered by OpenSSL 1.1
* Promise (np.promose.pas)
